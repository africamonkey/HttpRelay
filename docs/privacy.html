Privacy Policy

Last updated: April 2024

Information We Collect

HttpRelay is designed with your privacy in mind. The app does not collect, store, or transmit any personal information to external servers.

The following information is processed locally on your device:

  • Network Configuration: Server IP address and port number used for proxy functionality
  • Connection Logs: Temporary connection metadata (host, port, timestamp, status) stored locally for display purposes only
  • Usage Statistics: TX/RX byte counts maintained in memory during proxy sessions

Data Storage

All data processed by HttpRelay remains on your device:

  • Connection logs are stored temporarily and cleared when the app closes or when you clear the logs
  • Settings (such as your preference to show the tutorial on startup) are stored locally using iOS UserDefaults
  • No data is synced to iCloud or any other cloud service

Network Traffic

HttpRelay acts as a local proxy server on your iOS device:

  • The app does not intercept, inspect, or store the content of your network traffic
  • All HTTP/HTTPS data is forwarded transparently between your computer and the target server
  • The app does not add, modify, or filter any data passing through the proxy

Third-Party Services

HttpRelay does not use any third-party analytics services, advertising services, or data collection tools. The app has no network connections except:

  • Establishing the proxy server locally on your device
  • Connecting to target servers on behalf of your computer

Permissions

HttpRelay requires the following iOS permissions:

  • Local Network Access: Required to run the proxy server and establish connections to target servers
  • No other permissions are required

Children's Privacy

This app is not intended for children under the age of 13. We do not collect any information from children.

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top of this page. Any significant changes will be noted in the app's release notes.
